# TWINE
A python tool for imstall twine on termux automatically.

## Requirements
- python3 (3.11.4)
- Internet Connection 

## Module requirements 
- subprocess
- halo

## Install Termux
````
pkg up -y
pkg i git python -y
git clone https://github.com/illusionghost3/twine.git
cd twine
python twine.py
````
